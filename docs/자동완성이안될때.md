## 테일윈드 className='' <= 여기서 컨트롤+스페이스바 눌렀을때 자동완성가능한 리스트가 안뜰때

- 루트디렉토리에 .vscode폴더생성/settings.json안에 다음 복붙

```
{
  "tailwindCSS.experimental.configFile": "./src/styles/globals.css",
  "editor.quickSuggestions": {
    "strings": true
  }
}

```

## 요약

- 이 해결책이 효과가 있는 이유는 무엇입니까?

- 이전 버전의 Tailwind CSS에서는 이 설정을 사용하여 IntelliSense 구성 파일을 수동으로 지정했습니다. 그러나 Tailwind CSS v4에서는 VS Code의 Tailwind 확장 프로그램이 구성 파일을 자동으로 감지하므로 이 설정이 필요하지 않습니다. 이 설정을 유지하면 IntelliSense가 제대로 작동하지 않거나 불완전하게 작동할 수 있습니다.
